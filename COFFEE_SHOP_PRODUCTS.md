# Coffee Shop POS - Product List

## Overview
Your POS system has been updated with 90 coffee shop products across 8 categories.

## How to Update to Coffee Shop Products

### Option 1: Using the Reset Script (Recommended)
1. Close the POS application if it's running
2. Close NetBeans IDE
3. Run `reset_database.bat`
4. Start the application again

### Option 2: Manual Reset
1. Close the POS application
2. Delete the `pos.db` file from the project root
3. Run the application - it will create a fresh database automatically
