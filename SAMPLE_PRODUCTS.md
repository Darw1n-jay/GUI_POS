## Customization

### To Add More Products:
Use the MANAGE PRODUCT screen or run SQL:
```sql
INSERT INTO products (name, category, price, stock) 
VALUES ('Product Name', 'Category', 99.99, 50);
```

### To Add New Categories:
Simply add products with new category names:
```sql
INSERT INTO products (name, category, price, stock) 
VALUES ('New Item', 'New Category', 49.99, 30);
```

### To Clear Sample Data:
```sql
DELETE FROM products;
```
Then restart the application to reload sample data.
